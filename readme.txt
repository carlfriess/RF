RF library for ATMega and ATtiny

This library is designed to make it easy to use cheap RF links in projects
using both ATmega and ATtiny Microcontrollers.

Inspired by Kevin Darrah. http://www.kevindarrah.com

-----------------------------------------------------------------------------

USAGE

Connect TX modules to digital pin 2 and RX modules to digital pin 3.
